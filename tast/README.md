# محاكاة صفحة تسجيل الدخول لفيسبوك

هذا المشروع عبارة عن تطبيق ويب يحاكي صفحة تسجيل الدخول الخاصة بفيسبوك لأغراض تعليمية وتجريبية فقط.

## تحذير مهم

هذا المشروع مصمم لأغراض تعليمية وتجريبية فقط. تخزين كلمات المرور بهذه الطريقة لا يفي بمعايير الأمان المطلوبة في التطبيقات الحقيقية. في التطبيقات الحقيقية، يجب استخدام تقنيات التشفير المناسبة مثل bcrypt أو Argon2 لتخزين كلمات المرور.

## المكونات

المشروع يتكون من الملفات التالية:

1. `index.html` - صفحة HTML الرئيسية التي تحتوي على واجهة تسجيل الدخول
2. `styles.css` - ملف التنسيقات (CSS) الذي يضمن مطابقة التصميم لفيسبوك
3. `script.js` - ملف JavaScript للتعامل مع التحقق من البيانات والتفاعل مع المستخدم
4. `login.php` - ملف PHP للتعامل مع البيانات وحفظها
5. `data/` - مجلد يتم إنشاؤه تلقائياً لتخزين البيانات

## الميزات

1. واجهة مستخدم مطابقة لصفحة تسجيل الدخول في فيسبوك
2. التحقق من صحة البيانات المدخلة (البريد الإلكتروني/رقم الهاتف وكلمة المرور)
3. تخزين البيانات في ملف نصي بشكل آمن
4. حماية مجلد البيانات من الوصول المباشر
5. تصميم متجاوب يعمل على مختلف الأجهزة والمتصفحات

## التثبيت والتشغيل

1. قم بنسخ جميع الملفات إلى خادم ويب يدعم PHP (مثل Apache أو Nginx)
2. تأكد من تفعيل PHP على الخادم
3. تأكد من وجود صلاحيات الكتابة في مجلد المشروع لإنشاء مجلد البيانات
4. افتح المتصفح وانتقل إلى عنوان المشروع

## الأمان

يتم تطبيق إجراءات أمان أساسية في هذا المشروع:

1. تنظيف البيانات المدخلة لمنع هجمات XSS
2. حماية مجلد البيانات من الوصول المباشر عبر ملف .htaccess
3. تشفير بسيط لكلمات المرور قبل تخزينها
4. استخدام رؤوس HTTP الأمنية

ملاحظة: هذه الإجراءات ليست كافية للتطبيقات الحقيقية التي تتطلب معايير أمان أعلى.

## التخصيص

لتخصيص المشروع:

1. لتغيير طريقة تخزين البيانات، عدل ملف `login.php`
2. لإرسال البيانات عبر البريد الإلكتروني، قم بتفعيل قسم البريد الإلكتروني في ملف `login.php` وضبط إعدادات البريد
3. لتعديل التصميم، عدل ملف `styles.css`

## التوافق

المشروع متوافق مع:

- المتصفحات الحديثة (Chrome, Firefox, Safari, Edge)
- الأجهزة المختلفة (أجهزة الكمبيوتر، الأجهزة اللوحية، الهواتف الذكية)
- أنظمة التشغيل المختلفة (Windows, macOS, Linux, Android, iOS)

## المسؤولية

هذا المشروع للأغراض التعليمية والتجريبية فقط. لا تتحمل المسؤولية عن أي استخدام غير قانوني أو غير أخلاقي لهذا المشروع. يجب دائماً الحصول على موافقة صريحة من المستخدمين قبل جمع أي بيانات شخصية.

## الترخيص

هذا المشروع متاح للاستخدام التعليمي والتجريبي.
